selinux label  '-Z'
-------------
selinux label contains
1. user
2. role
3. type
4. level

example:
  - ls -Z
  - ps -Z
  - id -Z

unconfined_u:unconfined_r:unconfined_t:s0-s0:c0.c1023 = user:role:type:level
_u = user
_r = role
_t = type
last = level
